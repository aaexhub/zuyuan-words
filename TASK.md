# 单词记忆小程序开发任务

## 目标
为胡祖源同学开发一个网页版英语单词记忆小程序

## 功能需求

### 1. 单词学习
- 展示单词、音标、中文释义
- 支持语音发音（使用Web Speech API）
- 选择题测试：4个选项选正确的中文意思

### 2. 三色标记系统
- 🟢 绿色：已掌握（每周复习一次）
- 🟡 黄色：模糊（每天复习）
- 🔴 红色：不会（每天重点复习，自动加入错词本）

### 3. 错词本
- 自动收集所有标记为"红色"的单词
- 周末可以集中复习错词本里的单词
- 支持从错词本移除（标记为其他颜色）

### 4. 进度追踪
- 显示今日学习数量
- 显示本周正确率
- 显示各颜色单词数量

### 5. 词根提示（可选）
- 长词显示词根拆解
- 帮助记忆

## 单词数据

单词数据在 `../八上.txt`, `../八下.txt`, `../七上.txt`, `../七下.txt` 这四个文件中。
你需要：
1. 解析这些文本文件，提取单词、音标、中文释义
2. 生成一个 words.json 文件作为数据源
3. 如果解析困难，可以手动提取每册的关键单词（每册至少100个）

## 技术要求

- 纯前端实现（HTML + CSS + JavaScript）
- 使用 localStorage 保存学习进度和错词本
- 使用 Web Speech API 实现发音功能
- 响应式设计，手机和电脑都能用
- 界面简洁美观

```

```

## 用户体验

1. 首页选择要学习的册（七上/七下/八上/八下）
2. 进入学习模式，展示单词和选择题
3. 答题后选择：不会🔴 / 模糊🟡 / 会了🟢
4. 错词本页面可以集中复习红色单词
5. 进度页面显示学习统计

## 完成后

完成后运行以下命令通知我：
```bash
openclaw system event --text "Done: 单词记忆小程序开发完成" --mode now
```

然后启动一个本地服务器让用户可以访问：
```bash
cd ~/胡祖源/word-app && python3 -m http.server 8080
```
